== Description ==

Sermon'e is designed to help churches easily publish sermons online. is a new beautiful, modern sermon plugin that integrates seamlessly with any WordPress theme.

Would you like to see the plugin in action before installing it? We have a [demo website](http://159.89.194.38:8383/sermone/) showcasing all of it features!

= Powerpress Key Features =

* Video & Audio Sermons - Your media can be uploaded via WordPress or from popular sites like YouTube, Vimeo and SoundCloud.
* Filter by Series, Speakers, Topics & Books - Filter your sermons with our easy to use filter bar allowing you select your favorite series, speakers, topics, or books. 
* Easy To Use & Manage - Designed so that any staff member, volunteer, or paster can easily add and edit sermons and series.
* Archive support 2 layout -  Grid & List view, clean design that works with any WordPress theme.
* Social Share - Your visitors are only one click away from sharing their favorite sermons. Getting the word out has never been easier.
* Developer Friendly - There are many hooks available to customize Sermon’e to your needs.
* Powerful Shortcode - A full-featured shortcode system allowing you to show exactly what you need and where.
* Quickview Modal - allows you to quickly view a sermon, displaying a neat and smooth appearance
* Compatible with any Theme – We’ve worked hard to make sure Sermon'e is compatible with nearly every WordPress theme on the market.
* WordPress Widget - (Coming soon) allows to display the most recent sermons in the left or right sidebar.
* Gutenberg Block - (Coming soon) A full-featured block system allowing you to show exactly what you need and where.

== Frequently Asked Questions ==

= How do I get support? =
In the case that you need help with Sermon'e, you can submit a support ticket on our support page. We will make sure to get back to you as soon as we possibly can. Most likely the same day during our business hours.

= Will Sermon'e work with my theme? =
Yes! Sermon'e will work with any theme but may require some additional styling.

= Where can I report bugs? =
Report bugs on the Sermon'e [GitHub repository](https://github.com/Huynhhuynh/sermone/issues).

= Does sermon'e work on my WordPress multisite? =
Not yet as a network activated plugin, however you can install it individually on a multisite setup. We hope to work on network activated in the future.
